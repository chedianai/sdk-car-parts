# 车型库 SDK

>author JohnWang <wangjiajun@chedianai.com>

## 使用说明 ## 

此 SDK 仅限`车店AI`内部项目使用,请勿外泄;

### 配置 ### 

### Composer 引入 ###

```json
{
    "require": {
		"chedianai/car-parts-sdk":"~1.0"
    }
}
```
